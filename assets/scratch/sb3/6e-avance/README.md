# Structure .sb3 - 6e Avancé

## Contenu

Ce dossier contient la structure pour créer un fichier .sb3 pour la version avancée 6e (7 bugs).

## Fichiers

- `project.json` : Structure de base du projet Scratch (template minimal)

## Bugs inclus

Voir `../../source/6e-avance.md` pour la liste complète des 7 bugs.

## Création du .sb3

1. Créer une archive ZIP de ce dossier
2. Renommer en .sb3
3. Ouvrir dans Scratch

## Notes

⚠️ Le fichier `project.json` fourni est un **template minimal**. Pour un projet complet :

1. Créer le projet dans Scratch selon les instructions dans `../../source/6e-avance.md`
2. Exporter en .sb3 depuis Scratch
3. OU compléter le template JSON manuellement
