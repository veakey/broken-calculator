# Structure .sb3 - CM1 Basique

## Contenu

Ce dossier contient la structure pour créer un fichier .sb3 pour la version basique CM1 (1 bug).

## Fichiers

- `project.json` : Structure de base du projet Scratch (template minimal)

## Bugs inclus

Voir `../../source/CM1-basique.md` pour la liste complète des bugs.

## Création du .sb3

1. Créer une archive ZIP de ce dossier
2. Renommer en .sb3
3. Ouvrir dans Scratch

## Notes

⚠️ Le fichier `project.json` fourni est un **template minimal**. Pour un projet complet :

1. Créer le projet dans Scratch selon les instructions dans `../../source/CM1-basique.md`
2. Exporter en .sb3 depuis Scratch
3. OU compléter le template JSON manuellement
