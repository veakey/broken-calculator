# Activité Pré-Algorithmique - Grande Section

## Objectif pédagogique
Introduire les concepts de séquence et d'instruction de manière concrète.

## Matériel nécessaire
- Calculatrice de poche
- Cartes d'instructions
- Feuilles de séquence
- Jetons

## Activité

### Phase 1 : Suivre des instructions
1. Les enfants apprennent à suivre une séquence d'instructions
2. Exemple : "Appuie sur 4, puis sur +, puis sur 2, puis sur ="
3. Ils vérifient le résultat

### Phase 2 : Détection d'erreurs
1. L'enseignant donne une séquence avec une erreur
2. Les enfants identifient où se trouve l'erreur
3. Ils proposent la correction

### Phase 3 : Création de séquences
1. Les enfants créent leurs propres séquences
2. Ils les testent avec la calculatrice
3. Ils vérifient si tout fonctionne correctement

## Concepts introduits
- Séquence
- Ordre
- Erreur
- Correction

## Compétences développées
- Suivre des instructions
- Identifier des erreurs
- Créer des séquences
- Vérifier

## Prolongements
- Séquences plus longues
- Introduction de conditions simples
