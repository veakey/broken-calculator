# Activité Débogage Visuel - CE1

## Objectif pédagogique
Développer l'observation et l'identification visuelle de problèmes.

## Matériel nécessaire
- Version HTML simple de la calculatrice
- Feuilles d'observation
- Crayons de couleur

## Activité

### Phase 1 : Observation
1. Les élèves observent l'interface de la calculatrice
2. Ils notent tout ce qu'ils voient
3. Ils testent quelques calculs

### Phase 2 : Identification des problèmes
1. Les élèves créent une liste des problèmes observés
2. Ils classifient les problèmes (visuels, fonctionnels)
3. Ils priorisent les problèmes à corriger

### Phase 3 : Proposition de solutions
1. Les élèves dessinent une version corrigée
2. Ils expliquent leurs choix
3. Ils présentent leurs solutions

## Compétences développées
- Observation
- Analyse visuelle
- Communication
- Créativité

## Prolongements
- Tester les solutions proposées
- Créer de nouvelles interfaces
