# Activité Mini-Algorithmes - CE2

## Objectif pédagogique
Introduire les concepts d'algorithme et de séquence de manière simple.

## Matériel nécessaire
- Calculatrice de poche
- Cartes d'instructions
- Feuilles de séquence
- Diagrammes simples

## Activité

### Phase 1 : Algorithmes simples
1. Les élèves apprennent à créer des "recettes" pour des calculs
2. Exemple : Algorithme pour calculer 5 + 3
3. Ils testent leurs algorithmes

### Phase 2 : Détection d'erreurs
1. Les élèves reçoivent un algorithme avec des erreurs
2. Ils identifient les étapes incorrectes
3. Ils proposent des corrections

### Phase 3 : Création d'algorithmes
1. Les élèves créent leurs propres algorithmes
2. Ils les testent
3. Ils les améliorent

## Concepts introduits
- Algorithme
- Séquence
- Instruction
- Erreur

## Compétences développées
- Création d'algorithmes
- Détection d'erreurs
- Correction
- Test

## Prolongements
- Algorithmes avec conditions
- Diagrammes de flux simples
