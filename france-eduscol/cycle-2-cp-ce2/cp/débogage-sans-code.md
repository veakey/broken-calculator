# Activité Débogage Sans Code - CP

## Objectif pédagogique
Apprendre à identifier et corriger des erreurs de manière systématique.

## Matériel nécessaire
- Calculatrice de poche
- Tableau de résultats
- Feuilles de travail
- Crayons

## Activité

### Phase 1 : Test systématique
1. Les élèves testent la calculatrice avec une série de calculs
2. Ils notent tous les résultats dans un tableau
3. Ils comparent avec les résultats attendus

### Phase 2 : Identification des erreurs
1. Les élèves identifient tous les calculs incorrects
2. Ils cherchent des patterns dans les erreurs
3. Ils classifient les types d'erreurs

### Phase 3 : Correction
1. Les élèves créent un "carnet de réparation"
2. Ils notent toutes les corrections nécessaires
3. Ils vérifient leurs corrections

## Compétences développées
- Méthode de travail
- Observation
- Analyse
- Organisation

## Prolongements
- Créer des défis pour d'autres élèves
- Comparer différentes méthodes de correction
