# Tutoriel - Seconde Python Basique

## Objectif
Apprendre le débogage en Python en réparant une calculatrice.

## Prérequis
- Bases de Python (variables, fonctions, conditions)
- Utilisation d'un éditeur de code
- Exécution de scripts Python

## Étapes

### 1. Explorer le code
1. Ouvre `calculatrice.py`
2. Comprends la structure de la classe
3. Identifie les méthodes principales

### 2. Exécuter les tests
1. Lance le script : `python calculatrice.py`
2. Observe quels tests échouent
3. Note les erreurs

### 3. Identifier les bugs
1. Analyse le code de chaque méthode
2. Compare avec les résultats des tests
3. Identifie les lignes problématiques

### 4. Corriger
1. Modifie le code pour corriger les bugs
2. Sauvegarde
3. Relance les tests

### 5. Valider
1. Tous les tests doivent passer
2. Teste manuellement quelques calculs
3. Vérifie que tout fonctionne

## Conseils
- Utilise un IDE avec débogueur
- Ajoute des `print()` pour tracer l'exécution
- Corrige un bug à la fois
