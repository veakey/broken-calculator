# Tutoriel - CM1 Scratch Basique

## Objectif
Réparer une calculatrice créée avec Scratch (niveau basique).

## Instructions

### Étape 1 : Découverte
1. Ouvre le projet Scratch
2. Teste quelques calculs simples
3. Observe les erreurs

### Étape 2 : Analyse
1. Regarde les scripts
2. Identifie les blocs problématiques
3. Note tes observations

### Étape 3 : Correction
1. Modifie les blocs incorrects
2. Teste tes corrections
3. Vérifie que tout fonctionne

## Compétences développées
- Utilisation de Scratch
- Débogage simple
- Test et validation
