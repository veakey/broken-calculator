# Erreurs Simples - CM2

## Types d'erreurs à identifier

### Erreurs d'opération
- Opérations inversées
- Opérateurs incorrects

### Erreurs de logique
- Conditions incorrectes
- Variables mal utilisées

## Méthode de correction

1. Identifier l'erreur
2. Localiser dans le code
3. Corriger
4. Tester
