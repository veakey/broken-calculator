# Tutoriel - 5e JavaScript Basique

## Objectif
Apprendre les bases du débogage JavaScript.

## Étapes

1. **Explorer** : Comprendre la structure du code
2. **Tester** : Lancer les tests et observer les erreurs
3. **Identifier** : Trouver les bugs dans le code
4. **Corriger** : Modifier le code
5. **Valider** : Vérifier que tout fonctionne

## Conseils
- Corrige un bug à la fois
- Teste après chaque correction
- Utilise la console du navigateur
