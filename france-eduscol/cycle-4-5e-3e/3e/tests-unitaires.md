# Tests Unitaires - 3e

## Introduction aux tests

Les tests unitaires vérifient que chaque fonction fonctionne correctement.

## Structure

```javascript
function testAddition() {
    const result = testCalculate(5, '+', 3);
    const expected = 8;
    const passed = result === expected;
    // ...
}
```

## Types de tests

- **Unitaires** : Testent une fonction
- **Fonctionnels** : Testent un scénario complet
- **Intégration** : Testent l'ensemble

## Bonnes pratiques

1. Nommage clair
2. Un test, une vérification
3. Valeurs connues
4. Documentation

## Compétences développées
- Compréhension des tests
- Création de tests
- Validation
- Assurance qualité
