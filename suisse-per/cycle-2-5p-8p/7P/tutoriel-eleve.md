# Tutoriel Élève - 7P Scratch Version Intermédiaire

## Objectif
Réparer une calculatrice avec des bugs plus complexes dans Scratch.

## Instructions

### Étape 1 : Analyse du projet
1. Ouvre le fichier `calculatrice-erreurs.sb3`
2. Explore tous les scripts
3. Teste différentes opérations

### Étape 2 : Identification systématique
1. Crée un tableau avec :
   - Le calcul testé
   - Le résultat obtenu
   - Le résultat attendu
   - Le type d'erreur
2. Analyse les patterns d'erreurs

### Étape 3 : Correction méthodique
1. Corrige une erreur à la fois
2. Teste après chaque correction
3. Documente tes corrections

### Étape 4 : Tests finaux
1. Teste tous les types d'opérations
2. Teste avec des nombres décimaux
3. Vérifie la fonction "Clear"

## Compétences développées
- Débogage systématique
- Analyse de code
- Test et validation
- Documentation

## Validation
La calculatrice doit passer tous les tests sans erreur.
