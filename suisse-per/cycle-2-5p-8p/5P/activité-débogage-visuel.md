# Activité Débogage Visuel - 5P (PER Suisse)

## Objectif pédagogique
Apprendre à identifier visuellement les erreurs dans une interface.

## Matériel nécessaire
- Version HTML simple de la calculatrice (sans code visible)
- Feuilles de travail
- Crayons de couleur

## Activité

### Phase 1 : Test visuel
1. Les élèves utilisent la calculatrice en ligne
2. Ils notent les calculs testés et les résultats obtenus
3. Ils comparent avec les résultats attendus

### Phase 2 : Identification visuelle des problèmes
1. Les élèves observent l'interface de la calculatrice
2. Ils notent tout ce qui semble étrange ou incorrect
3. Ils créent une liste des problèmes observés

### Phase 3 : Proposition de corrections
1. Les élèves dessinent une version "réparée" de l'interface
2. Ils expliquent leurs corrections
3. Ils testent leurs idées si possible

## Compétences développées
- Observation attentive
- Identification de problèmes
- Communication visuelle
- Résolution de problèmes

## Prolongements
- Créer des interfaces avec des erreurs visuelles
- Tester les corrections proposées
