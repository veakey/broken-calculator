# Tutoriel Élève - 6P Scratch Version Basique

## Objectif
Réparer une calculatrice cassée créée avec Scratch.

## Instructions

### Étape 1 : Ouvrir le projet
1. Ouvre le fichier `calculatrice-erreurs.sb3` dans Scratch
2. Observe la calculatrice et teste quelques calculs

### Étape 2 : Identifier les erreurs
1. Teste les opérations suivantes :
   - 5 + 3
   - 10 - 4
   - 6 × 7
   - 20 ÷ 4
2. Note les résultats incorrects

### Étape 3 : Trouver les bugs
1. Regarde les scripts de la calculatrice
2. Identifie les blocs qui causent les erreurs
3. Note tes observations

### Étape 4 : Corriger
1. Modifie les blocs incorrects
2. Teste à nouveau les calculs
3. Vérifie que tout fonctionne

## Conseils
- Commence par les opérations simples
- Teste chaque correction immédiatement
- Demande de l'aide si tu es bloqué

## Validation
Tous les calculs doivent donner les bons résultats après tes corrections.
