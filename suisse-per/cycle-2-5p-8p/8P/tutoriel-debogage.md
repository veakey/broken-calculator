# Tutoriel Débogage - 8P Scratch Version Avancée

## Objectif
Maîtriser les techniques de débogage avec une calculatrice complexe.

## Méthodologie de débogage

### 1. Comprendre le problème
- Teste la calculatrice avec différents calculs
- Identifie tous les types d'erreurs
- Classe les erreurs par catégorie

### 2. Analyser le code
- Trace l'exécution des scripts
- Identifie les blocs problématiques
- Comprend la logique (même si elle est incorrecte)

### 3. Corriger méthodiquement
- Corrige une erreur à la fois
- Teste après chaque correction
- Documente tes changements

### 4. Valider
- Teste tous les cas possibles
- Vérifie les cas limites (division par zéro, etc.)
- Assure-toi que tout fonctionne

## Techniques avancées
- Utilisation de variables pour tracer l'exécution
- Tests unitaires avec des valeurs connues
- Documentation des corrections

## Compétences développées
- Débogage avancé
- Analyse de code
- Méthodologie de test
- Résolution de problèmes complexes
