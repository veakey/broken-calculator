# Activité Pré-Algorithmique - 3P (PER Suisse)

## Objectif pédagogique
Introduire les concepts de séquence et d'instruction sans programmation.

## Matériel nécessaire
- Cartes avec des instructions
- Calculatrice de poche
- Feuilles de séquence
- Jetons

## Activité

### Phase 1 : Séquence d'instructions
1. Les élèves apprennent à suivre une séquence d'instructions
2. Exemple : "Appuie sur 5, puis sur +, puis sur 3, puis sur ="
3. Ils notent les résultats obtenus

### Phase 2 : Détection d'erreurs dans la séquence
1. L'enseignant donne une séquence avec une erreur
2. Les élèves doivent identifier où se trouve l'erreur
3. Ils proposent la séquence corrigée

### Phase 3 : Création de séquences
1. Les élèves créent leurs propres séquences de calcul
2. Ils testent leurs séquences avec la calculatrice
3. Ils vérifient si le résultat est correct

## Concepts introduits
- Séquence d'instructions
- Ordre des opérations
- Détection d'erreurs
- Correction de séquences

## Compétences développées
- Suivre des instructions
- Identifier des erreurs
- Créer des séquences logiques
- Vérification

## Prolongements
- Séquences plus complexes
- Séquences avec conditions (si... alors...)
