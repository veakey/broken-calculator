# Activité Calculatrice Cassée - 2P (PER Suisse)

## Objectif pédagogique
Développer la pensée logique et la résolution de problèmes de manière concrète.

## Matériel nécessaire
- Calculatrice de poche
- Tableau de calculs avec erreurs
- Jetons ou objets pour compter
- Feuilles de travail

## Activité

### Phase 1 : Test de la calculatrice
1. Les élèves testent la calculatrice avec des calculs simples
2. Ils comparent les résultats avec leurs calculs manuels
3. Ils identifient les calculs qui donnent de mauvais résultats

### Phase 2 : Analyse des erreurs
1. Les élèves créent un tableau avec :
   - Le calcul effectué
   - Le résultat de la calculatrice (incorrect)
   - Le bon résultat
   - La différence
2. Ils cherchent des patterns dans les erreurs

### Phase 3 : Correction
1. Les élèves créent un "manuel de réparation"
2. Ils dessinent une calculatrice corrigée
3. Ils testent leurs corrections avec de nouveaux calculs

## Compétences développées
- Calcul mental
- Analyse de données
- Raisonnement logique
- Représentation visuelle

## Prolongements
- Créer des défis pour d'autres groupes
- Comparer différentes "calculatrices cassées"
