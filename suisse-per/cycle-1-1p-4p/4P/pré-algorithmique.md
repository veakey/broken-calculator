# Activité Pré-Algorithmique - 4P (PER Suisse)

## Objectif pédagogique
Développer la pensée algorithmique avec des séquences plus complexes.

## Matériel nécessaire
- Calculatrice de poche
- Cartes d'instructions
- Diagrammes de flux simples
- Feuilles de travail

## Activité

### Phase 1 : Algorithmes simples
1. Les élèves apprennent à créer des "recettes" pour des calculs
2. Exemple : "Pour calculer 5 + 3 :"
   - Étape 1 : Appuyer sur 5
   - Étape 2 : Appuyer sur +
   - Étape 3 : Appuyer sur 3
   - Étape 4 : Appuyer sur =
3. Ils testent leurs algorithmes

### Phase 2 : Détection et correction d'erreurs
1. Les élèves reçoivent un algorithme avec des erreurs
2. Ils identifient les étapes incorrectes
3. Ils proposent la version corrigée

### Phase 3 : Algorithmes conditionnels simples
1. Introduction du concept "si... alors..."
2. Exemple : "Si le résultat est plus grand que 10, alors afficher 'Grand'"
3. Les élèves créent leurs propres conditions

## Concepts introduits
- Algorithme
- Séquence d'instructions
- Conditions simples
- Détection d'erreurs

## Compétences développées
- Création d'algorithmes
- Détection d'erreurs
- Correction d'algorithmes
- Pensée conditionnelle

## Prolongements
- Algorithmes avec boucles simples
- Diagrammes de flux
