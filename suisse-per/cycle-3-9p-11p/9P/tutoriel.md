# Tutoriel - 9P JavaScript Débutant

## Objectif
Apprendre à déboguer du code JavaScript en réparant une calculatrice.

## Prérequis
- Connaissances de base en HTML/CSS
- Notions de JavaScript (variables, fonctions)
- Utilisation d'un éditeur de code

## Étapes

### 1. Explorer le code
1. Ouvre les fichiers `index.html`, `calculator.js` et `tests.js`
2. Comprends la structure du code
3. Identifie les fonctions principales

### 2. Lancer les tests
1. Ouvre `index.html` dans un navigateur
2. Clique sur "Lancer les Tests"
3. Observe quels tests échouent

### 3. Identifier les bugs
1. Regarde le code de `calculator.js`
2. Compare avec les résultats des tests
3. Identifie les lignes problématiques

### 4. Corriger
1. Modifie le code pour corriger les bugs
2. Sauvegarde et rafraîchis la page
3. Relance les tests

### 5. Valider
1. Tous les tests doivent passer
2. Teste manuellement quelques calculs
3. Vérifie que tout fonctionne

## Conseils
- Corrige un bug à la fois
- Teste après chaque correction
- Utilise la console du navigateur pour déboguer
