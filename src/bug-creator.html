Nouveau ? Consultez le Guide du Créateur de Bugs pour des instructions détaillées.

Sélectionner les bugs à introduire

Comment ça marche ? Cochez les bugs que vous voulez introduire dans votre calculatrice personnalisée. Cliquez sur "Générer le code" pour voir l'aperçu, puis "Télécharger" pour obtenir votre fichier. Remplacez ensuite le fichier src/js/calculator.js du projet par celui téléchargé.

Comment utiliser ?
  1. Cochez les bugs que vous voulez introduire
  2. Cliquez sur "Générer le code" pour voir l'aperçu
  3. Cliquez sur "Télécharger calculator.js"
  4. Remplacez le fichier src/js/calculator.js du projet par celui téléchargé
  5. Ouvrez index.html dans le navigateur - c'est prêt !
