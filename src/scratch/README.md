Calculatrice Scratch
====================

Ce dossier est destiné à regrouper toutes les versions **purement Scratch** de la calculatrice cassée :

- Pages HTML d'intégration ou de lancement des projets Scratch
- Documentation spécifique à l'utilisation des fichiers `.sb3`
- Scripts ou outils liés exclusivement à Scratch

Actuellement, aucune page HTML Scratch n'est définie dans `src/`. Elles pourront être ajoutées ici ultérieurement.
