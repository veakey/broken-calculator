Calculatrice Cassée - Scratch

Cette page est prévue pour accompagner une calculatrice cassée réalisée en Scratch (.sb3).

Comment utiliser cette version ?

Cette version ne contient pas de calculatrice en JavaScript, mais est pensée pour fonctionner avec un projet Scratch exporté au format .sb3.

Placez ici (dans ce même dossier ou un sous-dossier) vos fichiers Scratch de calculatrice cassée, puis ouvrez-les dans l’éditeur Scratch (en ligne ou hors ligne) avec vos élèves.

Vous pouvez aussi ajouter un lien direct vers votre projet Scratch en ligne :

  • soit en modifiant cette page pour insérer une iframe,
  • soit en ajoutant un simple lien vers l’URL de votre projet.
